# ESRTOS-project

##Obstacle avoidance Bot using FreeRTOS

###Hardware used

1.Arduino Uno 
2.Ultrasonic Sensor HC-SR04 
3.L298N Dual Channel Motor Driver 
4.Set of Brushed DC motor 
5.6s Battery (9 volts) 
6.Transistor battery (9 volts) 
7.Barrel jack male connector 

###Software used

1.Arduino IDE
2.FreeRTOS
